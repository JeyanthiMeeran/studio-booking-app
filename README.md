# Studio Booking App

## Overview
A React + Vite application to manage studio bookings.

## Features
- View bookings from JSON
- Filter bookings by status
- Add new bookings
- Form validation
- Modal popup for adding bookings
- Responsive layout

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Run Locally

npm install

npm run dev

## Project Structure
src/
- components/
- data/
- styles/

## Assumptions
- Bookings are stored in client-side state only.
- No backend or database is used.
